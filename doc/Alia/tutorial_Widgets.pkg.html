What is a widget?

A widget is a single reusable object-oriented user-interface element.

Widgets can, and often do, consist of many widgets. For example, APushButton is a widget - a button, and ALineEdit is also a wiget - a text entry field. ALoginWidget is a single widget that has a username and password field (ALineEdit widgets), and a submit button (a APushButtonWidget).
